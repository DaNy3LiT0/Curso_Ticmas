Sobre mi.

Me apasiona crear soluciones innovadoras y atractivas que combinen el arte y la tecnología. He trabajado en proyectos de diversos sectores, como educación, salud, entretenimiento, comercio electrónico, etc., tanto para clientes nacionales como internacionales.
Me considero creativo, proactivo, autodidacta y con capacidad para trabajar en equipo. Mi objetivo profesional es seguir aprendiendo y creciendo como diseñador gráfico y programador, aportando valor y calidad a cada proyecto que realizo.

  • Tengo conocimientos en herramientas de diseño como Photoshop, Illustrator, InDesign, Figma, etc., así como en lenguajes de programación como HTML, CSS, JavaScript, Python, etc.

Servicios.

Diseño Web

Diseño sitios web o aplicaciones que sean atractivos, intuitivos, responsivos y accesibles. Utilizo herramientas de diseño así como lenguajes de programación, para crear layouts e interfaces que faciliten la navegación y la experiencia de usuario.

Desarrollo Web

Desarrollo sitios web o aplicaciones que sean funcionales, seguros, escalables y optimizados para el SEO. Utilizo lenguajes del lado del servidor como PHP, Ruby, Python y Java, para crear el servidor, la aplicación y la base de datos que almacenen y procesen los datos de cada proyecto.

Branding

Creo marcas únicas y memorables que reflejan la personalidad y el valor de cada proyecto. Diseño la identidad corporativa, el nombre, el logotipo, el eslogan y la estrategia de comunicación que mejor se adapten a las necesidades y objetivos de cada cliente.
